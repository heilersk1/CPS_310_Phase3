Toy Store
Our Toy Store is a Django-based web platform that brings customers, employees, and mangers together in a single place. Each account type as its own entry point, views, and tasks so the right tools are always available
Managers
Accounts may only be created by another manager
Create, edit, and delete products
Create, edit, and delete events
Add and remove employees and other managers
Employees
Accounts may only be created by a manager
Edit and delete products
Edit and delete events
Customers
Create an account through the site
Browse products
Add products to cart
View events
